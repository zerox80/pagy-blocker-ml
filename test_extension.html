Chrome Extension Test

This page can be used to test if the extension loads properly.

  1. Open Chrome
  2. Go to chrome://extensions/
  3. Enable "Developer mode" (top right)
  4. Click "Load unpacked"
  5. Select this directory: /root/pagy-blocker
  6. The extension should load without errors

Expected Results:

  • Extension appears in the extensions list
  • No red error badges
  • Service worker shows as "running" or "inactive" (not "error")
  • Extension icon appears in toolbar
  • Clicking the icon opens the popup
